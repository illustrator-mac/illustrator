## 🛡️ Tips for Pro Workflows

1. **Use Cloud Documents for collaboration** — Real‑time co‑editing (like Google Docs but for vectors).
2. **Map a shortcut to “Save for Web”** — `⌥ + ⌘ + S` by default. Export PNG/SVG in one click.
3. **Enable GPU Performance** — *Illustrator → Preferences → Performance → GPU Performance* → ON.
4. **Set up Artboard presets** — Save Instagram, Twitter, YouTube templates as named presets.
5. **Learn the Pen tool** — Seriously. `P` key, practice for 2 hours, and your vectors will improve forever.
6. **Use Adobe Color** — Extract themes from images inside Illustrator (*Window → Color Themes*).
7. **Back up your workspaces** — `Window → Workspace → Export Workspace`. Save before updating.
8. **Try Firefly Text‑to‑Vector** — *Window → Text to Vector (Beta)* → Type “flat vector icon of a coffee cup” → instant editable vectors.

## 🔧 Common Issues and Fixes (Mac)

### ❌ “Illustrator is slow / laggy on my M2 Mac”
**Fix:** Go to *Preferences → Performance → GPU Performance* → make sure **GPU Performance** is ON. Also check *Preferences → Display → Anti‑aliasing* → set to "Optimize for Speed".

---

### ❌ “Fonts don’t appear in Illustrator (but they’re installed)”
**Fix:** Open *Font Book* → validate fonts. Then restart Illustrator. If that fails, clear font cache: `sudo atsutil databases -remove` (restart after).

---

### ❌ “Can’t save to iCloud Drive — permission error”
**Fix:** System Settings → Privacy & Security → Files and Folders → Adobe Illustrator → grant access to iCloud Drive.

---

### ❌ “Pen tool cursor is invisible or glitchy”
**Fix:** Toggle *View → Hide Edges* (`⌘ + H`) twice. Or disable Metal in Preferences → Performance → uncheck GPU Performance → restart.

---

### ❌ “Export for Web is missing PNG‑8”
**Fix:** Use **File → Export → Export As** → PNG → optimise manually. Adobe removed PNG‑8 from Save for Web in v27.

---

### ❌ “Illustrator crashes when using Variable Fonts”
**Fix:** Convert variable font to standard outlines: *Type → Create Outlines* (`⇧ + ⌘ + O`). Or update to v28.2 (above).

## ❓ FAQ

**Q1: Is Illustrator for Mac free?**  
> No, but Adobe offers a **7‑day free trial**. After that, it's $20.99/month for the single app or $54.99/month for Creative Cloud All Apps (Photoshop, After Effects, etc.). Students get ~60% off.

**Q2: Is Illustrator M1/M2/M3 native?**  
> **Yes** — since v26 (2022), Illustrator runs natively on Apple Silicon. It does **not** require Rosetta 2. Performance is significantly better than Intel version.

**Q3: Can I open Illustrator files on iPad?**  
> Yes — save as a **Cloud Document** (`.aic`) and open directly in Illustrator for iPad. Edits sync automatically.

**Q4: Does Illustrator work offline?**  
> Yes — you just need to sign in every 30 days. Without internet, you lose Cloud Documents and Adobe Fonts, but local files work fine.

**Q5: Can I run Illustrator on an old Intel Mac?**  
> Yes, v28 supports Intel Macs (2017+). But you lose some GPU acceleration and AI features. An M1 or newer is strongly recommended.

**Q6: What’s the difference between .ai and .aic?**  
> `.ai` = standard Illustrator file (local). `.aic` = Cloud Document (syncs, version history, co‑editing). Both hold identical vector data.
